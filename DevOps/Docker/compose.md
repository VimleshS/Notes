Multi-container apps are a hassle.

* Build images from Dockerfiles
* Pull images from the Hub
* Configure and create containers
* Start and stop containers
* Stream their logs

```
mysql:
  image: tutum/mysql
  environment:
    MYSQL_USER: root
    MYSQL_PASSWORD: ???
  volumes:
    - /mnt/sda1/var/lib/mysql_data:/var/lib/mysql
  ports:
    - "3306:3306"
web:
  build: .
  links:
    - mysql
    - mongodb
    - redis:redis
    - memcached
  volumes:
    - .:/var/www/webapp
  ports:
    - "80:80"
```
